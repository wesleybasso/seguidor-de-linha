# HDS-001 — PEGASUS Preliminary Datasheet

Documento preliminar.

## PEGASUS

Robô seguidor de linha modular.

## Placas

### HDB-001 Brain Board

- Black Pill STM32F411.
- XIAO ESP32-S3.
- SWD.
- UART.
- LEDs.
- Conectores para chassi.

### HDB-100 Chassis / Power Board

- LiPo 2S.
- XT30.
- TB6612.
- Buck 5V.
- Ventosa/sucção.
- Motores.
- Conectores J1/J2.

### HDB-101 Sensor Board

- Sensor frontal 8 canais.
- Laterais opcionais.

## Dimensões preliminares

- HDB-001: 75 x 40 mm.
- HDB-100: até 182 x 78 mm.
- HDB-101: ~90 x 16–20 mm.

## Status

Preliminar. Não usar para fabricação sem KiCad final e revisão.
